# 党建系统目录说明

当前仓库已经按用途拆成两套独立代码：

- `演示`
  - 存放旧静态演示版
  - 适合做页面展示、离线演示、GitHub Pages 静态发布
  - 主要目录：
    - `演示/release`
    - `演示/preview`

- `生成`
  - 存放真实数据版
  - 包含前端、后端、数据库接入、部署配置
  - 主要目录：
    - `生成/client`
    - `生成/server`
    - `生成/DELIVERY.md`

建议使用方式：

- 如果你要展示原型和静态页面：看 `演示`
- 如果你要继续做正式系统、连接后端和数据库：看 `生成`

当前部署建议：

- 演示版静态部署根目录：`coze版本党建/演示`
- 生产版前端部署根目录：`coze版本党建/生成/client`
- 生产版后端部署根目录：`coze版本党建/生成/server`
